# Guide: Words We Use

Use these names every time you build a feature or talk about the app. This keeps everything clear for the user.

### 1. The App and the Brain
- **Udyog Marg**: The name of this job search app.
- **Persona OS**: The "Second Brain" tool used for managing the business.

### 2. Key Features
- **The Pipeline**: The main screen where you track your jobs (Saved, Applied, etc.).
- **The Accelerator**: The onboarding screen that helps new users get started fast.
- **Job Importer**: The tool that pulls job details from LinkedIn or Indeed URLs.
- **Match Scoring**: The math that compares a Resume to a Job Description.
- **Interview Prep**: The tool that gives you questions and a pitch for your interview.

### 3. Rules for Using Words
- **Keep it Simple**: Never use a "fancy" word when a simple one works. (e.g., Use "Save" instead of "Commit").
- **Be Consistent**: Always call the job board "The Pipeline." Don't call it "The List" or "The Dashboard."
- **Talk Like a Human**: Write error messages and labels in plain, friendly English.
